Перед первым запуском приложения (далее - не потребуется) необходимо выполнить запрос лежащий в папке с приложением "software" - "Create_travel_firm_db.sql", выполняется он в программе
MS SQL Server Management Studio, так же необходима установка MS SQL Server.

1) Устанавливаем программное обеспечение скачать можно по ссылкам ниже или при помощи установщика лежащего в папке "SQLServer2017"
1.1) При установке выбираем тип - Базовая.
1.2) В конце установки выбираем - Скачать MSSMS(Microsoft SQL Server Management Studio) и устанавливаем её. Либо по ссылке ниже можно скачать эту программу отдельно.
2) Запускаем MS SQL Server Management Studio из меню Пуск. 
3) В открывшемся окне "Соединение с сервером" выбираем в строке "Имя сервера" - "Продолжить обзор", затем в "Ядро СУБД" выбираем свой Компьютер
4) Нажимаем кнопку - Соединить
5) Финальный шаг запускаем файл из папки с приложением - "Create_travel_firm_db.sql" и нажимаем зеленую кнопку Выполнить или кнопку F5.
6) Установка сервера выполнена и создание базы данных тоже. Можно запускать приложение
для первого входа используйте логин - admin пароль - admin

Ссылки на программное обеспечение для создания Базы Данных:

Before the first launch of the application (hereinafter, it will not be required), it is necessary to fulfill the request in the folder with the application "software" - "Create_travel_firm_db.sql", it is executed in the program
MS SQL Server Management Studio, installation of MS SQL Server is also required.

1) We install the software you can download from the links above or using the installer in the "SQLServer2017" folder
1.1) When installing, select the type - Basic.
1.2) At the end of the installation, select - Download MSSMS (Microsoft SQL Server Management Studio) and install it. Or from the link below you can download this program separately.
2) Launch MS SQL Server Management Studio from the Start menu.
3) In the "Connect to Server" window that opens, select "Server Name" - "Continue Browse" in the line, then in the "Database Engine" select your Computer
4) Press the button - Connect
5) The final step is to launch the file from the application folder - "Create_travel_firm_db.sql" and press the green Run button or the F5 button.
6) The server installation is complete and the database is created too. You can run the application
for first enter use login - admin password - admin
